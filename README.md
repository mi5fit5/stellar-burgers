## Описание проекта
Stellar Burgers — интерактивное веб-приложение космической бургерной. Проект реализован в рамках учебного курса от `Яндекс.Практикума`. В этом приложении любой пользователь может собрать свой уникальный бургер, оформить заказ и посмотреть ленту заказов в реальном времени.

В проекте реализован следующий функционал:
* **Навигация и маршрутизация:** Использование `react-router-dom` для переходов между страницами. Защищённые маршруты для неавторизованных пользователей;
* **Управление состоянием:** Глобальное состояние реализовано с помощью Redux (Redux Toolkit);
* **Авторизация и регистрация:** Полноценный флоу регистрации, входа, восстановления пароля и редактирования профиля;
* **Оформление заказов:** Создание заказа авторизованными пользователями с подсчётом общей стоимости;
* **Лента заказов и история:** Отображение всех заказов в приложении в реальном времени, а также просмотр личной истории заказов в профиле;
* **Модальные окна:** Просмотр деталей ингредиентов и информации о конкретном заказе через модальные окна;
* **Синхронизация с сервером:** Запросы к REST API для получения данных и WebSocket для обновления ленты заказов в реальном времени.

## Технологии
* React (функциональные компоненты, хуки)
* TypeScript
* Redux (Redux Toolkit, Redux Thunk)
* React Router DOM (v6)
* CSS Modules / БЭМ
* Webpack
* Storybook
* Cypress & Jest (тестирование)
* REST API и WebSockets

## Развёртывание и запуск

Для работы с проектом локально, выполните следующие шаги:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/mi5fit5/stellar-burgers.git
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd stellar-burgers
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Запустите проект в режиме разработки:
   ```bash
   npm start
   ```
   *Приложение будет доступно по адресу [http://localhost:4000](http://localhost:4000).*

5. Для запуска Storybook:
   ```bash
   npm run storybook
   ```
   *Storybook будет доступен по адресу [http://localhost:6006](http://localhost:6006).*
